# Rachel Garcia Author Website

Static author website for [rachel-garcia.com](https://rachel-garcia.com).

## Local preview

Run `start-preview.cmd`, then open `http://localhost:4173`.

## Cloudflare Pages

- Production branch: `main`
- Framework preset: `None`
- Build command: leave blank
- Build output directory: `stitch_rachel_garcia_author_website/site`
- Root directory: `/`

After the first deployment, add `rachel-garcia.com` and `www.rachel-garcia.com` as custom domains in the Pages project. The included `_redirects` file sends `www` traffic to the apex domain.
